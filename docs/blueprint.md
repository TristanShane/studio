# **App Name**: Chore Battle

## Core Features:

- User Authentication & Profiles: Secure sign-up and login capabilities using email/password or Google, alongside personal user profiles displaying a name and avatar.
- Household & Membership Management: Enable users to create new households or join existing ones via an invite code, with defined member roles (owner, admin, member).
- Chore Tracking & Management: Users can create, assign, claim, and mark chores as complete. Chores include a title, description, point value, frequency, due date, and status. Admins can approve or reject completed chores.
- Gamified Progress & Leaderboards: Implement a system to award points for completed chores, track user streaks, unlock digital badges for achievements, and display a competitive leaderboard for household members.
- Activity Dashboard & Notifications: A central dashboard view showing current tasks, user progress (points and streak), and recent household activity. In-app notifications will alert users about assigned or overdue chores.

## Style Guidelines:

- Color scheme: Light, inspired by the app's blend of playful competition and clean organization.
- Primary color: A vibrant yet grounding blue-purple for energy and reliability. Hex: #5540BF
- Background color: A very light, subtle purplish-grey to maintain a clean aesthetic. Hex: #F1F0F5
- Accent color: A brighter, clean blue for calls to action and highlights, creating good contrast. Hex: #5282E0
- Headline font: 'Space Grotesk' (sans-serif) for a modern, slightly techy feel.
- Body text font: 'Inter' (sans-serif) for its clean, objective readability across varying text lengths.
- Utilize modern, clean, and expressive icons that support the app's playful yet practical nature, with distinct designs for gamification elements like badges.
- A clear and hierarchical layout emphasizing content readability and user task flow, using ample whitespace for a clean and uncluttered feel.
- Subtle, smooth animations for transitions between screens and UI elements. Implement celebratory animations for chore completion, unlocking achievements, and point accrual.
